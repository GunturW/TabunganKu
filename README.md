# TabunganKu

<p>Aplikasi <strong>TabunganKu</strong> adalah sebuah aplikasi yang dapat merekam perjalanan uang anda, terdapat fitur yang dapat mencatat pemasukan dan pengeluaran yang anda lakukan
setiap saat. Anda juga dapat membuat rincian keterangan dan nominal pemasukan atau pengeluaran dan mengakumulasikannya sebagai catatan keuangan anda.<p>
<p>Fitur - fitur aplikasi <strong>TabunganKu</strong> : </p>
<ul>
  <li>1. Deskripsi dan nominal uang pemasukan atau pengeluaran.</li>
  <li>2. Percentase rincian masing-masing pengeluaran terhadap total pemasukan.</li>
  <li>3. Total Pengeluaran dan Pemasukan.</li>
  <li>4. Percentase total pengeluaran terhadap total pemasukan.</li>
  <li>5. Bulan dan tahun yang realtime.</li>
</ul>
<p><strong>Note : </strong>Aplikasi ini berbasis web yang dibuat dengan HTML, CSS, JavaScript dan responsive juga untuk Mobile atau Smartphone.</p>
